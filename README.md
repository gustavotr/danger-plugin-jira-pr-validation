> Import metadata from the issue on Jira and perform validations

## Usage

Install:

```sh
yarn add danger-plugin-jira-pr-validation --dev
```

At a glance:

```js
// dangerfile.js
import jiraPrValidation from 'danger-plugin-jira-pr-validation'

jiraPrValidation()
```
